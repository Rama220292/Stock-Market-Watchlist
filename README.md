# Project Brief
---
## Overview
This is my second project during the Software Engineering Bootcamp. There is 5 day timeline to develop the product from start to finish, and is being developed using JS, React and CSS. I landed on developing a simple application for a user to include stocks in his/her watchlist. 

## Timeline

Day 1: Project Planning (API, Wireframe, Components, Architecture)

Day 2: Setting Up API.Developing Components

Day 3: Developing Components

Day 4: Styling

Day 5: Testing, Deployment, Bug Fixes

---
## User Stories:
1. As a user, I want to search for a stock in a search bar using a stock ticker, and look at its latest price and 1-day return.
2. As a user, I want to include the stock that I have searched in my watchlist.
3. As a user, I want to remove a stock in my watch list.
4. As a user, I want to see a list of indexes, their prices, and 1-day return.
5. As a user, I want suggestions when I search for a stock. 

## Future Features
1. More detailed business information within each stock in a watchlist.
2. Expand to include stocks in SGX. 
3. Feature for users to include how many shares they own.
4. Dashboard to display the users' returns.
